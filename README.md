# DRF E-commerce

Este é um projeto de exemplo de uma API de e-commerce desenvolvida usando Django Rest Framework (DRF). 

A ideia é mostrar o uso de alguns pacotes que podem ser usados em projetos futuros:

- pytest
- pytest-django
- flake8
- django-mptt
- drf-spectacular
- coverage
- pytest-cov
- pytest-factorybot
